# אוצריא + AI של דיקטה

חבילת הרחבה לאוצריא ([Otzaria](https://github.com/Otzaria/otzaria)) שמוסיפה את כל כלי הבינה המלאכותית של [דיקטה](https://huggingface.co/dicta-il), עם דגש על מחשבים חלשים (i3 של אברך בכולל).

## מה כלול

| תיקייה | תוכן |
|--------|------|
| `docs/PLAN.md` | המסמך הראשי - ארכיטקטורה, החלטות, מפת דרכים |
| `docs/HARDWARE.md` | דרישות חומרה ואיזה מודל לטעון על איזה מחשב |
| `docs/INSTALL.md` | מדריך התקנה מעשי שלב אחר שלב |
| `dicta_service/` | שרת Python (FastAPI) שעוטף את כל מודלי דיקטה ב-REST API מקומי |
| `dart_client/lib/ai/` | קוד Dart לשילוב באוצריא - שירות, BLoC, מסכים |
| `scripts/` | סקריפטים: הורדת מודלים, המרה ל-ONNX, אריזה |

## איך זה עובד בקצרה

```
┌──────────────────────┐     HTTP localhost:7821      ┌─────────────────────────┐
│  אוצריא (Flutter)     │ ──────────────────────────►  │  dicta_service (Python) │
│  Dart UI              │                              │  FastAPI + ONNX/PyTorch  │
│  + AiService          │ ◄──────────────────────────  │  טוען מודלים בעצלתיים     │
└──────────────────────┘                              └─────────────────────────┘
                                                              │
                                                              ▼
                                                     ┌─────────────────┐
                                                     │ ~/.otzaria_ai/  │
                                                     │  models/  cache │
                                                     └─────────────────┘
```

השרת רץ ברקע בתוך תהליך נפרד שאוצריא מפעילה אוטומטית. הוא טוען רק את המודל שצריך כרגע (lazy loading) ופורק אותו אם לא משתמשים זמן מה - כדי לחסוך RAM.

## הכלים שנוספים

1. **חיפוש סמנטי** - לא רק התאמת מילים אלא משמעות
2. **הרחבת שאילתה אוטומטית** - "הלך" יחפש גם הולך/ילך/הליכה (לפי שורש)
3. **מציאת ציטוטים** - זיהוי אוטומטי של ציטוטי תנ"ך/חז"ל בתוך טקסט
4. **קישורים בין ספרים** - מציאת מקבילות בעלות משמעות דומה
5. **שאלות ותשובות** - "מי בנה את ירושלים?" → תשובה מהספר
6. **ניתוח דקדוקי** - שורש, חלק דיבור, מין, מספר לכל מילה
7. **זיהוי ישויות** (NER) - שמות חכמים, מקומות, ספרים
8. **ניקוד אוטומטי** - הוספת ניקוד לטקסט ללא ניקוד
9. **תרגום עברית-אנגלית** ולהפך
10. **השלמה אוטומטית חכמה** של ביטוי

ראה [`docs/PLAN.md`](docs/PLAN.md) לפירוט מלא.
